Git Assisment Hexa - Dec and Dec - Hexa calculator
Our code is made in order to create a base change for any given number between the Hexadecimal and Decimal bases both ways.

Instructions
To start the program the user must use idle or pycharm or any Python supported enviorment, and type starter() in order to summon the first function.
To exit press in the menu -1
As the menu suggests please use 1 to conver from decimal to hexadecimal and 2 for the other way.

The input must be related to each base and the programm will assert that.
